# Linux Admin Tasks using Cockpit (CLI Version)

This project contains command-line scripts for performing Linux system administration tasks inspired by Cockpit.

## Tasks Covered
- System Monitoring
- Service Management
- User Management
- Network Configuration
- Storage Management
- Firewall Configuration
- Web Server Setup

## Usage
Run each script:

bash setup.sh
bash services.sh
bash users.sh
...
