# RHCSA Lab 03 – Change Hostname

## Objective
Change the system hostname using the `hostnamectl` command.

This is a common task performed by system administrators when configuring new servers.

---

## Steps

1. Check the current hostname
2. Change the hostname
3. Verify the hostname
4. Confirm using hostname command

---

## Commands Used

```bash
hostnamectl

hostnamectl set-hostname station.domain40.example.com

hostnamectl

hostname
